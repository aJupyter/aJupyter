# 👨‍🎓 关于我
- 👋 大家好，我是南开大学（NKU）的研究生。
- 💬 我的研究兴趣和专注领域包括：大语言模型（LLM）、多模态大语言模型（MLLM）和增强检索生成（RAG）。
- 📖 我是 CSDN 人工智能领域的高质量内容创作者。
